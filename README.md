# Popmundo App Demo

An interactive mobile app concept for [Popmundo](https://www.popmundo.com) — the browser-based music simulation game. Built and designed by a long-time player.

**Live demo → [zenviolabs.github.io/popmundo-app-demo](https://bzynpb.github.io/popmundo-app-demo)**

## Design Principles

- **Feels like Popmundo** — same visual language: steel-blue header, gray sections, teal links
- **Web stays untouched** — desktop players keep their full experience
- **Community-first** — features chosen by a player who's been in the game for years
- **Lightweight** — HTML/CSS mockups, no framework, no build step
